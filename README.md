ioBroker enigma2 Adapter
==============
Adapter für ioBroker um Informationen von einem enigma2 Receiver abzufragen.


## Funktionen
- BOX_IP
- CHANNEL_SERVICEREFERENCE
- CHANNEL
- EVENTDESCRIPTION
- EVENTDURATION
- EVENTREMAINING
- HDD_CAPACITY
- HDD_FREE
- MESSAGE_ANSWER
- MODEL
- MUTED
- PROGRAMM
- PROGRAMM_INFO
- STANDBY
- VOLUME
